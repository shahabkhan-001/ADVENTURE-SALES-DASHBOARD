# Adventure Sales Dashboard

## 📊 Project Overview
This repository contains a comprehensive interactive dashboard built to analyze sales performance, revenue trends, and key business metrics for Adventure Sales. The goal of this project is to transform raw data into actionable insights for stakeholders to optimize sales strategies.

## 🛠️ Tools & Technologies Used
* **Power BI** (Specify which tool you used for the visuals)
* **Power Query** (For data extraction, cleaning, and transformation)
* **DAX** (For data modeling and calculating advanced business metrics)
* **Microsoft Excel** (Source data management)

## 🔑 Key Insights & Features
* **Revenue Analysis:** Tracks monthly and yearly revenue growth trends.
* **Product Performance:** Identifies top-selling product categories and sub-categories.
* **Customer Demographics:** Visualizes regional sales distribution and customer buying behavior.
* **Interactive Filters:** Allows users to dynamically slice data by region, year, and product category.
